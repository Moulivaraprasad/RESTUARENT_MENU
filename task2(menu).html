HTML Tag Dictionary

TAG NAMEDESCRIPTION
< !-- -- >This tag is used to apply comment in an HTML document.
< !DOCTYPE >This tag is used to specify the version of HTML.
A
< a >It creates a hyperlink or link.
< abbr >Defines an abbreviation for a phrase or longer word.
< acronym >Defines acronym for a word. (Not supported in HTML5)
< address >Defines the author's contact information of the HTML article.
< applet >Embeds a Java applet. (Not supported in HTML5)
< area >Defines the area of an image map.
< article >Defines self-contained content.
< aside >Defines content aside from main content. Usually a sidebar.
< audio >Embeds sound content.
B
< b >Makes text bold.
< base >Defines the base URL for relative URLs in the document.
< basefont >Sets default font, size, and color. (Not supported in HTML5)
< bdi >Provides text isolation for different text directions.
< bdo >Overrides the current text direction.
< big >Increases font size one level. (Not supported in HTML5)
< blockquote >Defines quoted content from another source.
< body >Defines the body section of an HTML document.
< br >Applies a line break.
< button >Represents a clickable button.
C
< canvas >Provides graphics space in a web document.
< caption >Defines a caption for a table.
< center >Aligns content in the center. (Not supported in HTML5)
< cite >Defines the title of a creative work.
< code >Displays code snippets.
< col >Defines a column in a table with shared properties.
< colgroup >Groups multiple columns in a table.
D
< data >Links content with machine-readable translation.
< datalist >Provides a list of predefined options for inputs.
< dd >Provides a definition in a description list.
< del >Defines text that has been deleted.
< details >Defines additional details the user can view or hide.
< dfn >Indicates a term being defined.
< dialog >Defines a dialog box or interactive component.
< dir >Container for directory list of files. (Not supported in HTML5)
< div >Defines a division or section.
< dl >Defines a description list.
< dt >Defines a term in a description list.
E
< em >Emphasizes text (usually italic).
< embed >Embeds external content like videos, PDFs, etc.
F
< fieldset >Groups related elements in a form.
< figcaption >Adds caption to a figure element.
< figure >Defines self-contained content.
< font >Defines font properties. (Not supported in HTML5)
< footer >Defines footer for a document or section.
< form >Defines an HTML form.
< frame >Defines a frame. (Not supported in HTML5)
< frameset >Groups frames. (Not supported in HTML5)
H
< h1 > to
Defines headings from level 1 to 6.
< head >Defines the head section of the document.
< header >Defines a header for a section or page.
< hr >Creates a thematic break (horizontal rule).
< html >Root element of an HTML document.
I
< i >Displays text in italic.
< iframe >Embeds another HTML document.
< img >Embeds an image.
< input >Defines input field.
< ins >Defines inserted text.
< isindex >Displays search field. (Not supported in HTML5)
K
Defines keyboard input.
L
< label >Labels form controls.
< legend >Caption for fieldset content.
< li >List item.
< link >Links to external resources (like CSS).
M
< main >Main content of the document.
< map >Defines image map areas.
< mark >Highlights text.
< marquee >Scrolling text/image. (Not supported in HTML5)
< menu >Defines menu list.
< meta >Defines metadata.
< meter >Defines a scalar measurement.
N
< nav >Defines navigation links.
< noframes >Alternate content for browsers not supporting frames. (Deprecated)
< noscript >Fallback content if script is not supported.
O
< object >Embeds an object.
< ol >Ordered list.
< optgroup >Groups options in a dropdown.
< option >Defines an option in dropdown.
< output >Shows result of calculation.
P
< p >Defines a paragraph.
< param >Defines parameters for an object.
< picture >Defines multiple sources for an image.
< pre >Preformatted text.
< progress >Progress indicator.
Q
< q >Defines inline quotation.
R
< rp >Fallback for ruby annotations.
< rt >Defines ruby text.
< ruby >Defines ruby annotation (pronunciation guide).
S
< s >Strikes through text (obsolete or incorrect).
< samp >Sample output from a program.
< script >Embeds JavaScript.
< section >Defines a document section.
< select >Dropdown menu control.
< small >Smaller text.
< source >Multiple media sources for media tags.
< span >Generic inline container.
< strike >Strikethrough text. (Deprecated)
< strong >Important text.
< style >CSS style definitions.
< sub >Subscript text.
< summary >Summary for details tag.
< sup >Superscript text.
< svg >Scalable Vector Graphics container.
T
< table >Defines a table.
< tbody >Groups table body content.
< td >Table data cell.
< template >Client-side template content.
< textarea >Multi-line text input.
< tfoot >Footer of a table.
< th >Table header cell.
< thead >Table header group.
< time >Time/date information.
< title >Document title (shown in tab).
< tr >Table row.
< track >Text tracks for media elements.
< tt >Teletype text. (Deprecated)
U
< u >Underlined text.
< ul >Unordered list.
V
< var >Variable name in programming/maths context.
< video >Embeds video.
W
< wbr >Suggests line break opportunity.
